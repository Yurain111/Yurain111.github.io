代码解释：

正则表达式 /[^\s\(\)]+?\((?R)?\)/

  • / ：正则表达式的定界符。
  • [^\s\(\)]+? ：匹配至少一个不属于空白符 ( \s )、左括号 ( ( ) 和右括号 ( ) ) 的字符（非贪婪匹配）。
  • \( ：匹配一个左括号。
  • (?R) ：递归调用自身，匹配嵌套结构（这是正则表达式的递归模式）。
  • \) ：匹配一个右括号。
  • 总体匹配的内容类似 foo(bar(baz)) ，即某种函数调用形式的嵌套结构。

preg_replace 会从 $code 中移除符合正则表达式的部分，即形如 函数名(参数) 的嵌套结构。例如：

  • 输入 $code 为 foo(bar(baz)) 时，移除后结果是空字符串 '' 。
  • 如果输入是 abc(xyz) + def(ghi(jkl)) ，移除后会留下 + 。
